### Задание 1

На лекции рассматривались режимы репликации master-slave, master-master, опишите их различия.

*Ответить в свободной форме.*

`Ведущий-Ведомый (Master-Slave): Это архитектура с единым источником данных. Данные записываются только на главный сервер (master), а затем подчиненные серверы (slaves) копируют эти данные и могут использоваться для чтения.`

`Двунаправленная репликация (Master-Master): Это архитектура, в которой каждый сервер выполняет роль как ведущего, так и ведомого. Запись может осуществляться на любой сервер. Затем данные реплицируются между всеми серверами.`
